# LuckyForge

AI 驱动的手机壁纸内容工厂。把"投喂参考图 → 风格提炼 → 提示词生成 → 批量出图 → 自动打分 → 素材打包"串成一条可反复运行的生产线，半自动化地产出同风格壁纸组（图 + 标题 + 标签），供人工在平台上点发布。

## 项目做什么

- 变现模式：内容工厂，自己运营平台账号发布
- 首版垂类：手机壁纸，架构预留向其他垂类（头像 / 海报等）扩展
- 趋势来源：人工投喂参考图，预留自动采集接口
- 发布方式：半自动，系统产出素材包，人工点发布
- 生图与分析：均依赖自部署的 chatgpt2api（gpt-image-2 出图、gpt-5 做风格提炼与打分）
- 产出形态：同风格壁纸组 + 标题 + 标签

## 技术栈

- 语言 / 运行时：Java 21（虚拟线程并发出图）
- 框架：Spring Boot 3.5.x
- 构建：Maven（`mvnw`）
- 工具库：Lombok
- 存储：首版文件系统 + JSON，后续按需引入 MySQL（元数据）与 MinIO（图片）

## 常用命令

```bash
# Windows
.\mvnw.cmd spring-boot:run
# macOS / Linux
./mvnw spring-boot:run

# 构建
.\mvnw.cmd clean package
# 测试
.\mvnw.cmd test
```

## 工作约定

- 用户语言偏好：中文沟通与注释。
- 代码注释：养成良好的中文注释习惯。
- 数据库约束：涉及表结构改动必须提供 SQL 脚本，不得通过 bash 直接向数据库插入数据，SQL 脚本交由用户手动执行。
- 变更流程：重大改动走 OpenSpec 提案（`openspec/`），提案内容尽量使用中文。
- 设计流程：`superpowers:brainstorming` 技能只做设计、不写文档，设计完成后再走 OpenSpec 提案流程。
